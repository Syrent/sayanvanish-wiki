# Raid Trigger

#### `FeaturePreventRaidTrigger`

| Parameter | Description                                    | Default Value | Possible Values |
| --------- | ---------------------------------------------- | ------------- | --------------- |
| `enabled` | Whether to enable the raid trigger prevention. | `true`        | `true`, `false` |
