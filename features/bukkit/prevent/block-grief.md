# Block Grief

#### `FeaturePreventBlockGrief`

| Parameter | Description                                   | Default Value | Possible Values |
| --------- | --------------------------------------------- | ------------- | --------------- |
| `enabled` | Whether to enable the block grief prevention. | `true`        | `true`, `false` |
