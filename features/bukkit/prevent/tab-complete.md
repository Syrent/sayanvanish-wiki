# Tab Complete

#### `FeaturePreventTabComplete`

| Parameter          | Description                                                                  | Default Value | Possible Values |
| ------------------ | ---------------------------------------------------------------------------- | ------------- | --------------- |
| `checkVanishLevel` | Whether to check the vanish level of the player before filtering completions | `false`       | `true`, `false` |
