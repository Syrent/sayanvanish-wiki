# Spawner Spawn

#### `FeaturePreventSpawnerSpawn`

| Parameter | Description                                     | Default Value | Possible Values |
| --------- | ----------------------------------------------- | ------------- | --------------- |
| `enabled` | Whether to enable the spawner spawn prevention. | `true`        | `true`, `false` |
