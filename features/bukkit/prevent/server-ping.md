# Server Ping

#### `FeaturePreventServerPing`

| Parameter | Description                                   | Default Value | Possible Values |
| --------- | --------------------------------------------- | ------------- | --------------- |
| `enabled` | Whether to enable the server ping prevention. | `false`       | `true`, `false` |
